Voguepay WooCommerce  plugin allows you to accept payment from local and international customers on your store.


== Description ==
VoguePay is a unique online payment processor whose vision is to offer buyers and sellers a secure and easy-to-use means of transacting business online.

VoguePay allows site owners to receive payment for their goods and services on their website without any setup fee.

VoguePay distinguishes itself from other online payment processors with its array of features that are guided by its core values:

* <strong>Security</strong>
* <strong>User Friendliness</strong>
* <strong>Reliability</strong>

To signup on Voguepay, visit  [here](https://voguepay.com/3445-0056682)


= Note =

Before you start accepting payment on your site, login to your [Voguepay Account](https://voguepay.com/login) go to <strong>Account Settings</strong> then <strong>API Settings</strong>, then turn on <strong>Enable Notification API</strong>. Then click on <strong>Save Changes</strong> for the changes to be effected. <br /><strong>It is very important that you do this before you start accepting payment on your site.</strong>


= Plugin Features =

* Demo mode
* Custom Plugin name and description
* Redirect or Inline payment methods
* Inline payment custom message
* Sending customer billing information during payment
* Setup extra charge as a fixed price or a percentage
* Support various languages
 
 
== Installation ==

= Installation =
* 	Login to your WordPress Admin area
* 	Go to "Plugins > Add New" from the left hand menu
* 	In the search box type "VoguePay WooCommerce"
*	From the search result you will see "VoguePay WooCommerce" click on "Install" to install the plugin
*	click "Activate" to activate it

= Note: =
If this is the first time of installing a WordPress Plugin, you may need to enter the FTP login credential information. This information is available through your web server host.


= Manual Installation =
1. Download the plugin zip file
2. Create a folder "woo-voguepay" in wp-content/plugins and extract all contents from the zip into it *or* upload the zip via the plugin section on the admin page.
3. Click installed plugins in plugin tab
4. Click activate to enable this on woocommerce  
5. Click settings under "VoguePay WooCommerce" to configure the plugin

= Configure the plugin =

* Voguepay Merchant ID: This can be found in your voguepay account

* Demo mode: Enable this for payment TESTING PURPOSE ONLY

* Payment Method: Inline method loads the checkout page directly on your website without redirection

* Inline Load Messsage: You can add a loading message for your customers while the checkout is being loaded.

* Customer Infomation: Send customer billing address and contact information. This helps in ensuring a successful transaction.
